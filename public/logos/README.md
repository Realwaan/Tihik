## Notes

- Some Brandfetch links may block direct anonymous downloads. If blocked, use the brand's official media kit/press page.
- Wikimedia file pages can be downloaded as SVG by opening the file page and choosing the original file.
- Keep filenames exactly as listed, or update mappings in `lib/wallet-badges.ts`.
